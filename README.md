# Albero di Anacleto

Progetto full stack per asilo / doposcuola con:
- sito pubblico
- login admin e docenti
- gestione personale
- gestione eventi/orari
- gestione avvisi
- database SQLite

## Avvio

```bash
npm install
npm run seed
npm start
```

Apri poi `http://localhost:3000`

## Accesso demo admin

## Note

Per la produzione conviene usare variabili ambiente, HTTPS e ulteriori protezioni di sicurezza.
